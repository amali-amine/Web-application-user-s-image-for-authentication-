Siamese Network in Javascript

Web authentification using user's image. This web application uses in the back-end resnet_v2_50 for face features extraction. In signing up, the user will provide an username and his picture, that they are going to be stored in some database (ex: PostgreSQL). In signing in, the app will compute the euclidean distance between the features of the new image of the webcam in the 'login' page and all features stored in the database. The app will authenticate the user only if the euclidean distance is smaller to a given treshold.

The login process will be published soon with an example of a database.
